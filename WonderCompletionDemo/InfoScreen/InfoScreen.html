Some info about this project.
The following techniques are used (with more or less extend) inside the app.
  • Cellediting
  • Custom Cells
  • Core Data with preloaded set
  • WebViews with templates
  • Asynchronous upload and download (simulated on secondary thread)
  • Delta updates on main thread
  • Version check on db
  • Audio signalling
All development material is provided in the project, this means, PSD files, creation tools, documentation tools, logic pro project, the complete git log (it is on bitbucket.org),
